# LOGIN_003 — Авторизация с неверным паролем

## Приоритет
High

## Функциональность
Авторизация пользователя.

## Предусловия
Пользователь зарегистрирован в Skyrexio.

## Тестовые данные
Валидный email пользователя и неверный пароль `12345678`.

## Шаги
1. Открыть страницу входа.
2. Ввести валидный email.
3. Ввести пароль `12345678`.
4. Нажать кнопку "Войти".

## Ожидаемый результат
Пользователь остается на странице входа. В правом углу окна отображается сообщение "неверный email или пароль".
