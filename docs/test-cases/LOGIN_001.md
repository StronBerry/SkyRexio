# LOGIN_001 — Успешная авторизация с переходом на главную страницу

## Приоритет
High

## Функциональность
Авторизация пользователя.

## Предусловия
Пользователь зарегистрирован в Skyrexio и имеет активные учетные данные.

## Тестовые данные
Валидный пользователь из локального конфигурационного файла `local.properties`.

## Шаги
1. Открыть страницу входа.
2. Ввести валидный email.
3. Ввести валидный пароль.
4. Нажать кнопку "Войти".
5. Дождаться загрузки главной страницы.
6. Проверить текущий URL.
7. Проверить отображение навигационного меню.
8. Проверить отображение блока "Статистика".

## Ожидаемый результат
Пользователь авторизован. Открыта страница `https://test.skyrexio.com/home`, отображаются пункты навигации и блок "Статистика".
